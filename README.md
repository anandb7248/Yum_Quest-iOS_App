# Yum_Quest-iOS_App

Spent majority of the current app development to set up the necessary frameworks of the proper classes and properties to represent each restaurants nearby the user as well as the specific menu items of the restaurants.

If you run the program and check the console, the output should display all the menu item's id, name, description, and price. All of this data is being received from the Foursquare API, which they use themselves use the Locu API under the hood.
